# Sales Performance Dashboard (Power BI)

This project presents an interactive Sales Performance Dashboard built using Power BI to analyze Actual vs Target sales.

## Project Overview
The dashboard helps track sales performance by comparing actual sales against predefined targets. It provides insights into monthly performance, salesperson contribution, and overall target achievement.

## Key Features
- KPI indicators for Total Actual Sales, Total Target Sales, Variance, and Variance %
- Monthly comparison of Actual vs Target sales
- Salesperson performance analysis
- Interactive filters for Team and Year
- Status indicator to identify whether targets are achieved

## Tools Used
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

## Insights
- Identify months where sales targets were not achieved
- Analyze top-performing salespeople
- Monitor overall sales performance against targets
